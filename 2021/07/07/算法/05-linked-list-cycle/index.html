05 环形链表

发表于 2021-07-07 分类于 算法

题目

链表的算法题解法都是围绕节点的 next 或 prev 指针，这道题也不例外。

解法一

通过哈希表辅助判断是否有环

解法二

通过快慢指针，如果到达链表尾部两指针还未相遇，则代表链表无环。

时间复杂度分别 O(n) 与 O(1)，快慢指针是一种思维技巧，学会即可。
