04 三数之和

发表于 2021-07-06 分类于 算法 阅读次数：

题目

今天这个题目有些难度，但是社区里有许多大神的经典解答可以参考学习。暴力解法就先不看了，时间复杂度最少要O(n^3)，先来看看双指针的解法。

解法

  • 首先需要将数组排序（升序），这个就是为了下面的工作铺垫
  • 设置一个固定位 k，然后设置两个指针 i 和 j，就有了 sum = nums[k] + nums[i] + nums[j] = 0
  • 判断 nums[k] 是否大于 0，因为数组是排序过的，如果 nums[k] > 0，则上面等式不会成立
  • 判断 nums[k] 是否等于 nums[k-1]，相等则跳过，题目要求答案不可重复

最后对 sum 的几种情况分别讨论
如果 sum < 0，则移动 i，同样因为数组是排序过的，只有移动左边，才会让 sum 更接近 0，在移动的过程中要注意去重
如果 sum > 0 同上
如果 sum == 0，则返回题解，注意去重
